# 源文件的字符编码

python 源文件的默认编码是 UTF-8，如果我们的 python 文件不使用默认编码，则需要在文件的第一行用注释声明文件的编码格式：

# -*- coding: 火星体 -*-

# 注释

既然上面提到了注释，在 python 中，可以用 # 声明注释

# 下面会输出一句真理
print("看这个文档的一定是个大帅哥或大美女！")

那除了用 # 进行注释外，我们还可以使用 ''' 或者 """ 注释代码，这两个通常用于写多行注释，比如在项目中说明某个工具的背景和具体用法：

'''
这是一个地图编辑工具，可用于编辑地图
作者：大橘
创作时间：2022 年 1 月 27 日
'''
class Tool:
    pass

# 代码换行

正常情况下，我们追求简洁的代码。但如果我们的某一行代码特别长，可以使用换行符反斜杠 \ 进行换行，比如下面这行计算每月剩下多少钱的代码就非常长：

"""
salary 工资
fang_dai 房贷
che_dai 车贷
wife_money 上缴给老婆的钱
parent_money 赡养父母的钱
children_money 养孩子的钱
eat_money 吃喝的钱
clothes_money 买衣服的钱
......
"""
my_money = salary - fang_dai - che_dai - wife_money - parent_money - children_money - eat_money - clothes_money

这代码太长了阅读起来的体验就不是很好，于是可以用反斜杠划分一下：

my_money = salary - fang_dai - che_dai - wife_money - parent_money - children_money - eat_money - clothes_money
